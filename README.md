# amyis.moe (WIP)

My portfolio site made with Nuxt 3

## Tech Stack

**Framework**: Nuxt 3

**Styling**: Tailwind CSS

**Request Client**: Fetch

**Package Manager**: pnpm

**Linter**: ESLint

**Formatter**: Prettier

**Typing**: TypeScript

## Running Locally

```bash
git clone https://github.com/amyisdev/amyis.moe.git
cd amyis.moe
pnpm i --shamefully-hoist
pnpm dev
```
