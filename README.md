# amyis.moe (WIP)

My portfolio site made with Vue.js

## Tech Stack

**Framework**: Vue.js

**Styling**: Tailwind CSS

**Request Client**: Axios

**Package Manager**: pnpm

**Linter**: ESLint

**Formatter**: Prettier

**Typing**: TypeScript

## Running Locally

```bash
git clone https://github.com/amyisdev/amyis.moe.git
cd amyis.moe
pnpm install
pnpm dev
```
